Taqbuct tafriqit n yiɣlanen juǧalef utseɛṭac.
Taqbuct tis snat n Lezzayer mbeɛd tesɛa uɛecrin yiseggasen n uraǧu.
Mbeɛd taqbuct-nni tamezwarut n useggas n alef utesɛemya utesɛin, ɛeddan tesɛa uɛecrin yiseggasen fell-as, i wakken sawḍen ɣer tis snat di juǧalef u tseɛṭac.
Tella-d temlilit n tagara n Teqbuct Tafriqit n yiɣlanen deg tedwilt-is tis tnin utlatin, di Lqahira tamanaɣt n tmurt n Maṣer.
Tebda temlilit-agi n tagara ɣef tmenya n tmeddit, i d-yessemlalen gar Terbaɛt taɣelnawt tazzayrit akked tin n Sinigal deg unnar Agraɣlan n Lqahira.
Yesekcem-d usleɣmay Belmadi deg timlilit-agi imurar yuɣen tanumi tturaren, mačči am terbaɛt n Sinigal i deg i d-yella ubeddel.
Urɛad mident snat ddqayeq mi iseffer unefray beddu n temlilit, yesaweḍ umurar Beghdad Boundjah ad d-yawi iswi awḥid n temlilit-agi.
Syin akkin, ikemmel uḥric amenzu n temlilit s uɣlab n terbɛat taɣelnawt tazzayrit, armi qrib d taggara n uḥric-a, i ten-teḥres cwiṭ terbaɛt n Sinigal.
Ikemmel uḥric wis sin n temilit s lḥers meqqren ɣef terbaɛt n yizzegzawen, maca ur ibeddel ara ugemmuḍ n temlilit-agi.
Ilmeẓyen n usleɣmay Belmadi, fkan-d yakk tazmert-nsen deg unnar, dayen i t-yeǧǧan ad seddun timlilt akken bɣan, yerna ad ḥadren ɣef yiswi-nsen awḥid.
Tesɛinn ddqiqa n wurar, ṣeɛbent i sanat n trebuyeɛ, yeqwa deg-sent yimenɣi ɣef ddabex gar yimurar ula gar yisleɣmayen n snat n trebuyaɛ-agi.
Tessuli tamlilit-agi n tagara s rrbeḥ n terbaɛt n yizzegzawen s yiwen yiswi ɣer ulac sdat n terbɛat n Sinigal i weɛren aṭas.
S wakka i d-rnan itri wis sin i terbaɛt-agi, mbeɛd timlilit ara yeqqimen deg umezruy.
Mi tfuk temlilit-agi, ttwaferqent tmidayin n ddheb ɣef yimurar d ugraw n terbaɛt taɣelnawt tazzayrit, syin akkin, tettunefk teqbuct-agi i umurar Riyad Maḥrez d yimeddukkal-is.
Am wakken, daɣen i tettunefk teqbuct n umurar Igerrzen i Benacer.
Ad d-nesmekti kan belli tarbaɛt taɣelnawt tazzayrit terbeḥ akk timlilyin-is i setta n teqbuct-agi, yerna d tarbaɛt i d-yufraren si tama n temḥaddit, imi ala sin n yiswiyen i d asen-d-swan.
Ulamma aṭas n yiwudam n waddal afriqi, ur yerǧin ara Izzegzawen ad awḍen armi d tagarant n teqbuct-agi.
Maca beggnen-d ayen nniḍen i mi i d-wwin amecwar yessewhamen si tazwara n teqbuct-agi, ladɣa mi rnan Cot divoire d Nijirya, snat-agi n terbuyaɛ i d-ssumren ad awint taqbuct-agi tafriqit n yiɣlalen.
Iberreḥ usleɣmay n terbaɛt taɣelnawat Jamel Belmadi, yenna-d: D timlilit n umezruy, i mi d tagi i d taqbuct tamzwarut ara nerbeḥ berra n tmurt-nneɣ, mbeɛd tinna n tesɛin di tmurt-nneɣ.
Nesɛa tamurt i d-yefkan aṭas n yimurar, s wawal nniḍen d tamurt n ddabex uḍar.
Ulamma teṣɛeb temlilit-agi, maca nessaweḍ ad neɣleb yiwet n terbaɛt meqqren, isem-is Sinigal, yuraren taqbuct tamaḍlant iɛeddan.
Ikemmel yenna-d: Tajmilt ad tuɣal i yimurar i d-yefkan nnig tezmert-nsen i tmurt-nsen, mebla nutni nekk ulac-iyi, i mi lliɣ kan deffir-sen d amnir, ttwelliheɣ-ten daya.
Ɣer taggara maḍi, ad d-nesmekti belli tarbaɛt-agi taɣelnawt, tettraǧu-tt Teqbuct gar Imenẓawen deg useggas n juǧalef u waḥe uɛcrin ɣer tama n Teqbuct Tamaḍlant deg useggas n juǧalef utnin uɛcrin, afud igerrzen sya ɣer sdat.
