Mass Aselway, iɛeggalen n tesqamut inebgawen n lɛez d lḥerma. 
Ayen i aɣ-d-ijemɛen ass-a, d tasalalt n tezrawt n turagt. 
I d-nheyya i mu nefka azwel, aserwes gar tmeslayt n At Mḥend tasaḥlit, d tmeslayt n At Seɣrucen. 
At Seɣrucen n Waṭlas Alemmas n Merruk. 
Asentel-nneɣ yerza taɣult n tesnilest deg yiger n tesnantala. 
Deg uswir n tesnalɣa i yerzan anagraw n umyag. 
Yerzan tasnilest timerwest. 
Iswi agejdan d anadi ɣef ugmmuḍ n usmeskel n wass-a. 
Deg snat n tmeslayin-a yeggugan i ttekkan ɣer yiwet n tutlayt. 
Deg snat n tmura yemgaraden. 
Aseqsi agejdan: D acu yezdin d wacu yemgaraden gar snat n tmeslayin-a. 
Iwakken ad nesɛu tamuɣli ɣef tmukrist-a. 
I d-yeqqaren ma yella tantaliwin n Tefriqt Ugafa d yiwet n tutlayt? 
Neɣ yal yiwet d tutlayt iman-s ad d-yeglun s usezdi d useggem n tutlayt Tamaziɣt. 
Turdiwin: Ma yella yiwet n tutlayt iman-is!
